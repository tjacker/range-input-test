Testing Instructions

iOS Testing

  1. Open this page in Safari on iOS 15+
  2. Test each slider by tapping and dragging
  3. Note any issues with touch responsiveness
  4. Test in scrollable container (Test Case 12)
  5. Try different step values (Test Cases 2-4)
  6. Test range slider with both handles (Test Case 5)

Screen Reader Testing

  1. Enable your screen reader (VoiceOver, NVDA, JAWS)
  2. Tab through each slider
  3. Listen to announcements (value, range, label)
  4. Test keyboard navigation (arrows, Page Up/Down, Home/End)
  5. Verify formatted values are announced correctly
  6. Test range slider with distinct handle labels

For detailed testing procedures, see .kiro/specs/ngx-slider-feature-parity/MANUAL_TESTING_GUIDE.md
