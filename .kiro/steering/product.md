# Product Overview

This is an Angular application demonstrating a custom range input component with advanced features. The primary focus is on creating reusable, accessible form controls with custom value formatting and visual feedback.

## Inspiration

This project replicates the behavior of [ngx-slider](https://github.com/angular-slider/ngx-slider) but uses **native HTML range inputs** instead of a custom implementation. This approach provides:

- Better accessibility out of the box
- Improved mobile compatibility
- Reduced complexity and maintenance
- Native browser optimizations

Not all ngx-slider features are needed (e.g., vertical orientation is excluded).

## Key Features

- Custom range slider component with configurable display options
- Number formatting with prefix/suffix support (e.g., percentages)
- Progress bar visualization
- Tick marks with customizable steps
- Debounced drag and drop events
- Scenario-based slider implementation for financial modeling
- Custom value accessor pattern for Angular forms integration
- Native HTML range input foundation
